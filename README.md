<h1>F1 History Viewer</h1>
This repository contains an React + TypeScript application that allows users to view and explore visualized historic data from Formula 1.<hr>
<h3>What this app does?</h3>
This app proviedes an interactive platform to explore statistics from Formula 1 and its history. It allows for comprehensive access to Formula 1 statistics spanning more than 75 years of racing history. By presenting the data through charts and visual analytics, it enables users to explore data in more friendly way. The application supports quick and efficient data analyzing, while also allowing users to revisit their recent searches.<hr>
App is made of eight pages:

 - Home
 - Seasonal Results
 - Driver Stats
 - Constructor Stats
 - Track Stats
 - Compare Drivers
 - Pit Stops
 - Season Calendar

<b>Home</b> - Allows user to view last race results, current championship standings for both Driver's and Constructor's championship and also current season's progress.<br>
<b>Season Results</b> - Browse complete results of any selected season (from years 1950-current) including driver and constructor standings with race-by-race results.<br>
<b>Driver Stats</b> - View individual driver's career stats with viewing his stats like seasons raced, total number of wins, world championship titles, and per-constructor stats.<br> 
<b>Constructor Stats</b> -  Lets you analyze team history including seasons raced, total wins, worlds championship titles and driver's contributions.<br>
<b>Track Stats</b> - Discover circuit informations with last race results, race history, top winners on selected track and view of location.<br>
<b>Compare Drivers</b> - Allows user to create a Head-to-head comparision between two selected drivers with charts.<br>
<b>Pit Stops</b> - Analyze pit stop statistics from 2011 onwards, including fastest/slowest stops and driver rankings.<br>
<b>Season Calendar</b> - View race schedules for any season with dates.<br>

<h3>How to run this application</h3>
Prerequisities:

 - Node.js (v18 or higher recommended)
 - npm 

<b>Installation</b>

 - Clone the repository
 
 <code>git clone https://github.com/yourusername/f1HistoryViewer.git
<br>cd f1HistoryViewer</code>
 - Install dependencies
 
 <code>npm install</code>
 - Start the development server

 <code>npm run dev</code>
